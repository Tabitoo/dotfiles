
1. En la pagina de hostinger ir a la opcion de vps
2. En la pantalla del vps, ir a la opcion que dice Terminal, esto va a abrir una nueva pestaña, en la cual nos tenemos que loguear
3. La primer opcion va a ser para escribir el username, ahi tiene que poner root y dar enter
4. Luego va a pedir la contraseña, ahi tiene que poner ------ y darle enter, en este campo no se va a ver lo que escribe pero eso es normal
5. una vez logueado, escriba el siguiente comando cd aprendeAstrologia/packages/api, esto lo va a llevar a la carpeta en donde esta el backend.
6. Cuando este en este directorio, escribir el siguiente comando pm2 stop 0
7. Luego escribir el siguiente comando: pm2 start yarn -- start, con este comando se levanta la api. Luego de esto ya tendria que estar levantado el backend, puede tardar un minuto. Recordad utilizar estos comandos de pm2 dentro de la carpeta aprendeAstrologia/packages/api
