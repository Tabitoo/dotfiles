
## Que es WhoIs

Whois es tanto un protocolo como una base de datos publica que contiene información sobre las personas/entidades que poseen un dominio o ips.

El tipo de información que suele contener estas bases de datos sobre una persona es la siguiente:

- Datos del titular: Nombre, correo, dirección, etc.
- Fecha de creación y vencimiento del dominio.
- Registrador: Empresa que gestiono el dominio.
- Estado del dominio.

Estas bases de datos son útiles para temas de ciberseguridad, ya que con la información que brinda podríamos detectar si un sitio es malicioso o no, o si un dominio fue robado recientemente. Gracias a los datos que brinda podemos hacer un seguimiento para determinar estas cosas.

### ¿Cómo se compone WhoIs?

WhoIs es un protocolo y una red distribuida de bases de datos cada una administrada 
por distintas organizaciones dependiendo del TLDs (.com, .ar, .net, etc).

Estas organizaciones operan bajo la coordinación de ICANN, el organismo global que regula el sistema de nombres de dominio e IPs a nivel mundial.

En nuestra región tenemos la organización LACNIC (**Registro de Direcciones de Internet de América Latina y el Caribe**)  la cual se encarga de administrar y registrar las direcciones IPv4 e IPv6 de toda América Latina.

Luego tenemos NIC Argentina (Dirección Nacional del Registro de Dominios de Internet) la cual es una organización argentina encargada de administrar y registrar los dominios .ar de nuestro país, en caso de que queramos consultar la información de un dominio .ar esta es la base de datos WhoIs que tenemos que consultar
